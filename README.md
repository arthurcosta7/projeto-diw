# 🍽️ Restaurantes de Belo Horizonte

## 📌 Descrição

Projeto de desenvolvimento de um site informativo sobre restaurantes de Belo Horizonte.

A proposta é apresentar diferentes opções gastronômicas da cidade, com foco em organização, design e experiência do usuário.

---

## 🎓 Contexto

Este projeto está sendo desenvolvido como parte da disciplina de Desenvolvimento de Interfaces Web da faculdade.

O site será construído ao longo de todo o semestre, acompanhando a evolução dos conteúdos aprendidos em aula.

---

## 🚧 Status do projeto

⚠️ Em desenvolvimento

Atualmente, o projeto conta apenas com estrutura em HTML e estilização em CSS.
Novas funcionalidades e melhorias serão adicionadas progressivamente.

---
